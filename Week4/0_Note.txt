그래프 탐색 알고리즘 (graph traversal)
    Breadth_first search (BFS) 넓이 우선 탐색
    Depth_first search (DFS) 깊이 우선 탐색

    알고리즘 설명:
    BFS
    1. 어떤 정점을 선택해 방문한 정점을 큐에 push함,
    2. 큐가 비어 있지 않을때까지 반복
        매 반복마다 하나의 정점을 큐로부터 가져옴 (front)
        해당 정점과 연결된 정점들 가운데 아직 queue에 들어가지 않은 정점을 방문
        방문하면 해당정점에 visit 표시

        그런데 큐에 한번도 들어가지 않은 vertex가 존재하면 그래프는 연결되어 있지 안핟고 할 수 있음

    DFS
    위와 동일하나 큐 대신 스택을 사용


알고리즘 구현 (수도코드)
    vertex, edge 개수 정보 필요
    각 정점과 연결된 변의 정보가 linked list 형태로 필요요
    그래프의 정점을 담는 클래슷 생성
