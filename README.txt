강의명: 인공지능을 위한 알고리즘과 자료구조: 이론, 코딩, 그리고 컴퓨팅 사고(KMOOC - 성균관대 허재필 교수님)
강의 내용: 자료구조와 알고리즘
학습 기간: 2021.07~2022.01
학습 내용: 강의를 통한 자료구조와 알고리즘 이론 복습(현재 2회독중) 및 백준 실전 문제 풀이

2회독 계획
1. 백준 merge sort tree 풀기
2. heap 자료구조 및 정렬 documnet 참고해서 구현하기
3. quick_sort 구현하기
4. dp 부분 복습할 때 행렬곱 다시 공부
5. 2회독은 거꾸로 DP, 최단경로 알고리즘, 최소신장트리 알고리즘, 서로소 집합(크루스칼, 프리즘) -> 이후 순서대로

- DP 그대로, 최단경로 알고리즘 서로소 집합은 부르투 포스
- 힙은 우선순위 큐, quick은 구현만
- Merge는 플래티넘 도전 및 분할정복
- 비선형 부분 BFS
- 주말 파싱 및 시물레이션 풀기
