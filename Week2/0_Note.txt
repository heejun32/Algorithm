연습문제 5 * 5 행령 가운데 0 기준 왼쪽 오른쪽 -1, 1 출력하기

내 코드:

rows = 5
columns = 5
matrix = [[0 for col in range(columns)] for row in range(rows)]

for row in range(rows):
    for column in range(columns):
        if row == column:
            matrix[row][column] = 0
        elif row > column:
            matrix[row][column] = 2
        else:
            matrix[row][column] = 1
    print(matrix[row])
